# 全局服务 Staff 管理界面

Linux 全局数据服务提供只读 Staff 管理界面：

```text
http://服务器地址:7778/staff
```

页面展示：

- 已接入联机模式的玩家 UID、用户名、Windows 实例 ID、注册和最近活动时间。
- 每位玩家的服务器存档数量、军团身份数量和排行榜记录数量。
- 当前军团等级、经验、团长、成员、成员贡献和待处理申请。
- 排行榜编号、参与记录数量、最高分和最近更新时间。

页面默认每 15 秒自动刷新，也可以手动刷新和按 UID、用户名、昵称或实例 ID 搜索玩家。

## 接口

```text
GET /api/staff/overview
```

该接口只返回管理元数据，不返回完整服务器存档内容和存档校验值。

当前研究环境未启用身份验证，因此访问服务器 `7778` 端口的设备均可打开 Staff 页面。
